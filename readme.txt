# Example project using renv and git.

It involves creating an R project called renv-example and ticking the git and renv options.

I inlcuded an .Rmd file that loads some packages and saves a lock.file, just as an example of the kind of workflow that might be used.

It basically follows suggestions found here for renv and git, respectively:

https://kevinushey-2020-rstudio-conf.netlify.app/slides.html#146

https://journals.sagepub.com/doi/full/10.1177/2515245918754826

Author: Richard Ramsey
